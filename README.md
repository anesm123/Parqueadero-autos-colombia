# 🚗 Sistema de Información Parqueadero Autos Colombia

## 📌 Descripción

Este proyecto corresponde al desarrollo de un sistema de información para la gestión del parqueadero **Autos Colombia**, enfocado en el control de las operaciones diarias como:

- Registro de entrada y salida de vehículos
- Gestión de usuarios
- Administración de celdas
- Gestión de pagos de mensualidad

El desarrollo se realizó por iteraciones, aplicando conceptos de Ingeniería de Software.

---

## 🧩 Funcionalidades del sistema

### 🔹 Iteración 1: Gestión de vehículos
- Registro de vehículos
- Control de entrada y salida

### 🔹 Iteración 2: Gestión de usuarios y celdas
- Registro de usuarios
- Asociación de vehículos a usuarios
- Registro de celdas
- Asignación de celdas a vehículos

### 🔹 Iteración 3: Gestión de pagos
- Registro de pagos de mensualidad
- Consulta del estado de pago
- Actualización del estado de pago
